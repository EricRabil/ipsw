The BIG one (4.7G) is the file system (aka the one that has the dyld_shared_cache).

The other two are the arm64eCustomerRamDisk and the arm64eUpdateRamDisk, however, you cannot mount them as they are actually im4p files. üòï

You can now extract the Img4 payloads with the following command:

‚ùØ unzip -p iPhone_4.7_14.0_18A5319i_Restore.ipsw 038-44087-104.dmg > 038-44087-104.dmg
‚ùØ ipsw img4 extract 038-44087-104.dmg
   ‚Ä¢ Parsing Im4p
      ‚Ä¢ Exracting payload to file 038-44087-104.dmg.payload

Rename the payload back to a DMG

‚ùØ mv 038-44087-104.dmg.payload 038-44087-104.dmg.payload.dmg

And now you can open the üÜï DMG to mount the RAM disk image.
